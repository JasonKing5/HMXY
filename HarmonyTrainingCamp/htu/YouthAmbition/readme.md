# 少年志（诗词应用）
> 河师大教学随堂项目。
> 
> 劝君莫负少年志 曾许人间第一流
---

### 简要描述：

1. 引导页 `Index.ets`
   1. 倒计时跳转`主体展示页`
   2. 跑马灯文字
   3. `logo`动画（角度和透明度）
2. 主体展示页 `MainPoem.ets`
   1. 顶部标题栏：`logo`+`名称`+`设置齿轮`
   2. 轮播区域
   3. 展示诗词：
      1. 点击诗词区域发送网络请求更新展示内容
      2. 点击`Toggle`进行`收藏`/`取消`
      3. 点击星星跳转`收藏页`进行查看收藏列表
3. 收藏页 `CollectPoem.ets`
   1. 顶部标题：`返回箭头` + `名称`
   2. 收藏内容
      1. 未收藏过，提示信息
      2. 收藏过，按分类进行列表分组展示
      3. 删除某一条收藏数据
         1. 如果删除的数据是当前 `主体展示页MainPoem.ets` 正在展示的数据，取消主体展示页的`Toggle`状态。
         2. 如果删除的数据不是当前 `主体展示页MainPoem.ets` 正在展示的数据，主体展示页的`Toggle`依旧是选中状态
