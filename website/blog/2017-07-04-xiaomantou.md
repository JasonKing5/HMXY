行走江湖，总是会被骗的，然后成长。

---

<!--truncate-->
今年的七月初搬进红专路42号的房子里，附近的每一条街我都走过无数遍，重新走过，勾起从前的点滴回忆。

   <br/>

讲一件多年前被骗过的小事。

   <br/>

那是许久之前的某个冬天，我还是个刚读大学的小少年，彼时的生活还按照时间节点来数日子——暑假，寒假，开学，期末。

   <br/>

那个寒假的气温真的很冷，那个寒假的经历真的很温暖。

 
  <br/>
文彤彤约我去陪她办一件小事，大概离家不算远，去人才市场一趟。具体是替朋友交党费还是转什么关系或文件，我已然记不大清楚。

   <br/>

到现在重新走一遍，距离确实算不得远，从刘叔叔的小店走去人才市场，不过几个小路口而已。
  <br/>
 

那天我和她走出人才市场的大门，冬日的风吹到脸上有些凌厉。她撩了撩彼时尚未留长的发，告诉我想吃黄河路东明路一件糕点店里的蜂蜜小馒头，大概是烤箱烘烤出来的面包，内里包裹着红豆，底层有厚厚的蜂蜜，做成馒头形状，吃起来香甜可口。
  <br/>
 

沿着黄河路往前走，胸科医院的门口聚集着一群围观的人，当中坐着一农民工，面前摆放着一件古董。

   <br/>

风很冷，两点钟的阳光看起来有些无力，街上的人都裹着羽绒服。车马喧嚣的城市按部就班地在这个钢铁森林里上演着冷漠。

   <br/>

农名工盘着腿在人行道旁的暗黄色地砖上席地而坐，大约五十多岁的老人，脸上是岁月雕刻的皱纹，头发有些窝囊，身上是破旧且单薄的脏衣，脚上一双黄胶鞋沾满了干下来之后的泥土。他的双手有明显的皲裂，红色血迹的裂口塞着工地上干活留下的黄泥。农名工被围观地有些紧张，脸上是尴尬的表情，双手来回搓着大鱼迹。

 

文彤彤蹲在人群中看了一会农民工，表情有些心疼，把我的手握的更紧了一些。她又看向农名工面前摆放的那枚古董。

   <br/>

那是一枚晶莹剔透的大块玉石，经能人巧匠打磨出精美的模型，便是一枚栩栩如生的大乌龟，四条腿和尾巴脑袋都很精致，更出彩的是大乌龟的龟壳顶上正中央是一枚硕大的寿桃，桃尖有点泛红，简直能够和真正的熟透了的蟠桃以假乱真了。

还有更出彩的呢。蟠桃的一周是八个造型各异的小人，神采表情仿若真人，衣襟繁华，褶皱清晰。文彤彤看得入神，对这样的奇珍异宝很是惊奇，眼神里透着赞为观止的欣赏，她把头转向左边，问我这些小人是什么意思？

我有告诉她，这是八仙过海。你看拿拐杖的这个，他是个瘸子，叫铁拐李。旁边的是吕洞宾，就是俗话说狗咬吕洞宾不识好人心的那个神仙，这个很美的女子是何仙姑。

当我巴拉巴拉地逐个介绍完，文彤彤的眼神中尽是佩服的喜爱了。

文彤彤又问我这个大乌龟是什么意思？

我有告诉她，从造型看，这个应该取名叫做：八仙祝寿龟。因为这里有个寿桃。乌龟代表着长寿。

   <br/>

文彤彤随口问了我一句：你怎么懂这么多！

明明是疑问句，说出来确实感叹的口吻。这句话在此后的多年里，每当我讲出有趣且动听的故事。她就会对我讲一遍。然后我就开始沾沾自喜地得意了。

 
  <br/>
那天的我们蹲在人群中看路人和农名工的讨价还价。战况很是激烈，有人出价到180，农民工有明显的不甘心。总觉得这是件宝贝，怎么着也能卖到230。可是农名工实在不善言辞，反复重复着那句：添点吧。再添点。

这句河南话的意思很直白，无非就是多给点的意思。从那个农名工的口中讲出来，有些乞讨的语气。
  <br/>
 

其中有个戴眼镜的大叔，从外貌来看大约是个文化人，砍价最凶，想要买下来的意向也最明显。最后拉扯了许久，出价到两百块。实在不能再多。

那个不善言辞的农名工又实在不甘心，可是他当真不懂营销，翻来覆去的那句：再添点吧。

眼镜大叔的气势有些咄人，自作主张要出两百块钱买回去。把两张毛爷爷甩在农民工的膝盖上便伸手去抢那一枚八仙祝寿龟。

   <br/>

农民工急切地向前扑倒，双手紧紧抱着他的宝贝。有些结巴地向这个世界宣言：

我在工地挖出来的，就想卖个钱吃顿饭。把欠工友的两百块钱还上。你就给两百块钱，我还了钱就吃不上饭。我一整天都没有吃饭。

  <br/>

农民工结结巴巴地说完以后，眼角有浑浊的湿润。

眼镜大叔收起地上的两张毛爷爷，有些犹豫要加点钱还是就此离去。

  <br/>

我听到文彤彤小声地嘀咕：再添点吧，再添点吧！

我感受到文彤彤把我的手掌握地更紧一些了。

   <br/>

眼镜大叔最终还是决定收起两百块钱走人。

农名工有些失落，当他看到眼镜大叔站起身将要离去的时候，突然松开紧紧捂住的那个沾满泥土的大乌龟。

   <br/>

农名工声音虽然高了几个分贝，却显得更加狼狈：行行行！两百也卖！

  <br/>

当时的我由心里发出一声冷笑。开始感叹这个农民工的演技真棒，简直可以去拿奥斯卡了。

   <br/>

在我的认知里，在街头摆上一件古董，不论摆的是什么，随便做旧，粘上些泥土。再编出一套自己在工地干活时候挖出来的说辞。忽悠个几百块钱便是赚到了。

因为现代科技的成熟，随便做出些工艺品的模具，以树脂浇灌塑型，成本最多几十块。拿到街头充当古董，骗到即是赚到。

  <br/>

大学时候宿舍里有个室友，是我的四哥，闲着没事就爱看寻宝节目，我也跟顺上两眼。学得些鉴定真假的不入流的细枝末叶。尽管如此，我看一眼那个毫无雕琢痕迹的工艺品，还是能断定，这就是一个树脂浇灌的假货。

  <br/>

我还在心里嘲讽那个愿意花两百块买一个工艺品的傻逼眼镜大叔，突然就听到文彤彤的声音：我买！我添钱！
  <br/>
 

我下意识地去制止文彤彤的冲动，抬头就看到她眼角有泪流出来。

我知道，文彤彤在心疼农名工的辛酸了。我还知道，文彤彤对社会的认知度向来很低。

   <br/>

那天我没有制止文彤彤的行为，反倒做了一个实实在在的傻逼。

我们俩把身上的钱如数拿出来，凑上文彤彤手包里的零钱。一共两百七十四块。

文彤彤把钱交给农民工以后，人群就开始散去，我抱着那个大约有两三斤重的八仙祝寿龟陪着她沿着黄河路向东走。
  <br/>
 

文彤彤扣了扣何仙姑衣袍上的泥土，又摸了摸圆润的仙桃。她仰着脸问我：两三百块呢，值不值呀？

   <br/>

我双手托着那枚大乌龟的底座，毫无玉质的手感。当即编出一套天衣无缝的说辞来。

我有告诉她：非常值得呀，如果是明代宣德年间的，差不多价值百万，回头我们卖出去，郑州的房子都买的起呢！

我还告诉她：即便是假的古董，就冲这两三斤的玉石，论斤卖也能值得个三五百块！

我有想起文彤彤把两百七十四块钱塞给农名工的时候祝福过他：天气太冷，去吃碗热面条吧。

我还告诉文彤彤：即便这个东西一文不值，我们帮了一个凄惨的农名工还了工友的欠款，还请他吃一顿热饭暖身体，即便算不得什么善举，也算一件好事了。多做好事，福未至，祸已远。

文彤彤很努力地点点头，也安慰我：等过几年，我们大学毕业工作了，等我们结婚了，等奶奶八十大寿了，你就把这个送给奶奶做寿礼。奶奶肯定很开心。

文彤彤之所以安慰我，是因为那两百七十四块钱里有一大半是她从我钱包里掏出来的。
  <br/>
 

后来我们毕业了，奶奶八十大寿也快到了。我很祈祷文彤彤的奶奶能够长命百岁，身体安康。可是我们没有结婚，我也没有资格去送贺礼了。

即便是送贺礼，即便八仙祝寿龟是真的古董，我想也是不能送这个的。因为奶奶和刘叔叔都是虔诚的基督教信徒。怕是跟中国的神仙有所冲突的。

 
  <br/>
那天我抱着八仙祝寿龟，文彤彤抱着我的胳膊，沿着黄河路往东走，我觉得被文彤彤抱着的那个冬天很温暖。

 
  <br/>
在黄河路东明路的糕点店门口看了一会蜂蜜小馒头，可是我们身无分文。彼时还没有微信和支付宝。

所以我欠下了一份蜂蜜小馒头。

 
  <br/>
后记

文彤彤上过很多当，也被骗过许多钱。我亦是教会过她许多社会认知。

但是那天我没有拆穿那个骗局。因为我知道，当我们都长大成人后，怕是再也不会有当初心疼陌生人的悸动了。当文彤彤足够成熟的生活，怕是不会为路人流眼泪了。

但是当文彤彤在农名工大叔的面前流下眼泪时候，我看到的不是她的傻，而是善良。

文彤彤生性怯懦，当她在人群中大声喊出要买下那个工艺品的时候，我理解的不是冲动，而是勇敢。

所以我喜欢这样善良勇敢的人儿。我愿意被这样俗套的情节骗上一次。

 

  <br/>

现在，那个八仙祝寿龟在驻马店中华路西段租的廉价房的床板夹层里蒙上了厚厚的灰尘。

原来我们的一生，善于隐藏掉许多东西，以为可以就此忘却。殊不知，越是不去触碰，越是有利了保鲜。

  <br/>

我该如何隐藏自己，才能巧妙地度过下半生？

   <br/>

——2017年7月初
