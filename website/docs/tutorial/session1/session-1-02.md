# 卸载软件

> 安装失败原因很多人是因为网络不稳定。不要使用校园网（不稳定）或者内网（需要代理）。

如果安装失败，可以卸载重装。推荐好用的卸载软件：[geek-最好用的卸载工具.exe](https://mayw-teaching.feishu.cn/file/PzOSbHyZOoObtmxEAeKcxgZ3nhg?from=from_copylink) （6.3M）
