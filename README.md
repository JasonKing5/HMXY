# HMXY

本项目为[鸿蒙学院网站](https://hm.codefe.cn/)项目源代码。

## 项目信息
- **开发者**: 庄生
- **维护者**: 帝心

## 结构概览
项目主要包含以下部分：
- 博客文章（blog/）
- 文档教程（docs/）
- 配置文件（config/）
- 源代码（src/）
- 静态资源（static/）
- 脚本工具（scripts/）

## 文档教程
项目包含丰富的文档教程，主要分为以下几个部分：
- **HarmonyOS 教程**: 提供从环境搭建到高级功能的详细指导。
- **AGC 教程**: 介绍如何使用 AGC（AppGallery Connect）进行开发。
- **开发指南**: 包含本地运行、开发指南、添加文档等内容。

## 博客文章
博客文章主要涵盖以下主题：
- **人生杂谈**: 分享个人感悟和生活故事。
- **HarmonyOS**: 提供 HarmonyOS 的视频教程和学习资料。

## 技术栈
项目使用以下技术栈：
- **前端**: React, TypeScript, Docusaurus
- **后端**: Node.js, Express
