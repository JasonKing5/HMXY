## 项目演示效果

> 1. 开屏倒计时进入
> 2. 主页tabs展示 `何浅红` + `帝心推荐`
> 3. 何浅红：
>    1. 点击主题更换诗词
>    2. 点击收藏按钮收藏当前诗词
>    3. 点击查看收藏按钮跳转页面
> 4. 帝心推荐
>    1. 展示推荐列表
>    2. 点击诗词进行收藏
>    3. 再次点击取消收藏
